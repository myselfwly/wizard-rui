# 基于 Antd 组件封装声明

> 声明：  
> 就单纯的个人玩一玩不喜就当看笑话，喜欢的可以跟着学习下思路和培养下习惯。

> 背景：  
> antd 组件常见功能的易用性并不是很好，主题适配也不是很直接，决定走一套自己的 UI 库，试着玩一玩。

> 思路：  
> 主要分为两个方向。1. 用法封装，提高易用性，进而提高开发效率；2. 主题系统，提高主题适配的效率，主要还是开发效率和学习成本。主题，目前还没想好是做成系统还是说是工具。总之先做先预留功能，事上练吧。
